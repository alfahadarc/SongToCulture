Dataset profile

Last.fm Dataset

Listener-generated tags and song-to-song similarity data aligned to MSD tracks, plus detailed genre annotations—ideal for understanding perceived style and building tag-aware recommenders.

Scope & Coverage

  • Song-level tags from the Last.fm community mapped onto MSD track IDs.
  • Similarity relations between songs, enabling graph-based navigation and clustering.
  • Segment-level genre annotations to anchor style labels in the audio timeline.

Used for

  • Genre tagging benchmarks or zero-shot classification using crowd-applied labels.
  • Building similarity graphs to evaluate collaborative filtering or content-tag hybrids.
  • Validating audio features against listener perception with the segment genre cues.

Attributes

msd_track_id tags (with weights) similar_songs genre_annotations

  • tags: User-generated tags with associated weights indicating relevance.
  • similar_songs: List of MSD track IDs deemed similar by Last.fm users.
  • genre_annotations: Time-aligned genre labels for segments within each track.
