Dataset profile

Lakh MIDI Dataset (LMD)

A large collection of MIDI files across genres and eras. The LMD-matched subset lines up with MSD entries, giving symbolic note/chord sequences you can align with audio features.

Scope & Coverage

  • Diverse MIDI files with melodic, harmonic, and arrangement detail.
  • LMD-matched subset links each MIDI file to a corresponding MSD track ID.
  • Symbolic representation allows clean separation of notes, chords, key, and instrument parts.

Used for

  • Compositional analysis: chord progressions, melodic contour, or instrument roles.
  • Cross-modal experiments: align symbolic sequences with MSD audio or lyrics.
  • Training data for generation models that need structured note/chord events.

Attributes

msd_track_id (matched subset) note events chord labels key instrument program

  • note events: Time-stamped note on/off events with pitch, velocity, and duration.
  • chord labels: Chord annotations aligned to the MIDI timeline.
  • key: Estimated musical key of the piece.
  • instrument program: MIDI program numbers indicating instrument types used.
